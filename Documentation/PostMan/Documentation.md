# Documentation for the video Uploader

## Upload Video

- **URL** - http://oluwashenor-001-site1.ftempurl.com/api/uploadVideo
- **Method** - POST
- **Include Header** 
    ```
    Content-Type - multipart/form-data
    ```
- **Body(form-data)** 
  ```
  files - (attach your file to this key)
  ```
- **N.B File Cannot be larger than 50MB**
